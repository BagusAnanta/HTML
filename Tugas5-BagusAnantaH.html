Pengenalan HTML

HTML (Hypertext Markup Language) adalah bahasa markup standar untuk membuat halaman website. Bahasa ini berfungsi untuk membuat struktur website hingga menyusun format teks dan gambar pada halaman web.

Biasanya, dalam penggunaan HTML pengguna harus menuliskan tag-tag yang sesuai, dalam penulisan tag diawali tanda kecil "<" dan ditutup dengan tanda lebih besar ">", dan tag html memiliki tag pembuka dan tag penutup.

Walaupun HTML bukan bahasa pemrograman, Anda wajib mempelajarinya jika Anda ingin menjadi web developer yang andal. Di artikel ini,kami akan menjelaskan secara lengkap apa itu HTML, fungsinya, komponennya, dan cara kerjanya.

Dasar HTML

Tag judul (Tag "hn")

Tag "h" merupakan tag yang dimana berfungsi sebagai pembuat judul pada HTML, jumlah (n) pada HTML terdiri dari <"h1"> hingga <"h6">. Tag <"h1"> digunakan pada judul pada level pertama. Lalu tag lainnya digunakan pada sub heading atau level berikutnya. Masing-masing judul akan ditampilkan dengan ukuran teks yang berbeda.

Paragraf (Tag "p")

Paragraf merupakan kumpulan beberapa kalimat. pada web, paragraf biasanya digunakan untuk menampilkan tampilan berupa teks atau artikel.

Catatan : Paragraf pada html disimbolkan dengan tag <"p"><"/p">

- Tag "br"

Tag br merupakan tag yang berfungsi untuk membuat baris baru pada paragraf.

Catatan : br pada html disimbolkan dengan tag <"br"><"/br">

- Tag "hr"

Tag hr merupakan tag yang berfungsi sebagai membuat garis lurus secara horizontal. Dan biasanya digunakan untuk memisahkan beberapa konten atau paragraf

Catatan : hr pada html disimbolkan dengan tag <"hr"><"/hr">

- Tag "pre"

Tag pre merupakan tag yang digunakan untuk menampilkan teks atau paragraf dalam format yang sudah kita tentukan di HTML

Catatan : pre pada html disimbolkan dengan tag <"pre"><"/pre">

List & Tabel

Tabel

Pada HTML, kita dapat membuat tabel untuk menyampaikan informasi berupa data berbasis baris dan kolom, didalam HMTL dalam pembuatan tabel terdapat beberapa tag yang dapat digunakan untuk membuat tabel

  1. Tag <"table"> untuk membungkus tabel

  2. Tag <"thead"> untuk membungkus bagian kepala tabel

  3. Tag <"tbody"> untuk membungkus bagian body dalam badan tabel

  4. Tag <"tr"> (tabel row) untuk membuat baris

  5. Tag <"td"> (table data) untuk membuat sel

  6. Tag <"th"> untuk membuat judul pada header

List

Pada html, kita dapat membuat list karena di HTML sudah tersedia macam-macam tag untuk membuat list, terdapat 3 macam list :

  1. Ordered List, merupakan list yang terurut (pada HTML menggunakan tag <"ol">)

  2. Unordered List, merupakan list yang tidak berurut yang menggunakan simbol-simbol pada itemnya (menggunakan tag <"ul"> dan <"li">)

  3. Description List, merupakan list yang berisi deskripsi atau penjelasan dari sesuatu

Multimedia

Multimedia merupakan penggunaan media pada komputer yang menyajikan dan menggabungkan teks,suara,gambar,animasi,audio,dan video. Dengan alat bantu (tool) dan tautan (link) sehingga pengguna dapat melakukan navigasi,berinteraksi,berkarya, dan berkomunikasi.

Pada HTML, kita dapat memasukan media berupa foto atau gambar, dan pada html pula kita dapat memasukkan audio hingga dapat memutar video pada antarmuka web yang kita buat, dengan syarat file musik,video atau gambar harus menjadi satu dengan file projek HTML kita.
